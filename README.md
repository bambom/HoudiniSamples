## 作者有话说

一颗做独立游戏的心愈来愈燥热难耐，但对于个人来说，游戏开发所需要的美术资产实在是让人难以承受，遂想尝试使用Houdini来做程序化离线生成来减少美术成本。



本仓库大致分为两个部分，

1. 作者学习其他人的Houdini经历与工程文件，以及必要的解释说明

2. 作者本人的一些作品



希望本仓库能帮助到一些Houdini同好，共勉！
